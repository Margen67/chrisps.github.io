Just enter a floating point value you want to manipulate in the input box. Below you may write javascript that manipulates the binary format of the value using the variables mantissa, exponent, sign, uvalue, and value.

The initial input value is readable as a constant named originalValue. Value corresponds to the floating point value with the current mantissa, sign and exponent, and is updated after each statement that writes to the float's components as well as after a write to uvalue. The float's components are also updated by writes to value and uvalue. uvalue corresponds to the binary representation of value/the union of mantissa, sign and exponent

Additionally, the function log(string) is predefined for writing to the output window.
